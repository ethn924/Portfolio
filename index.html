01 — À propos

Qui suis-je ?

Je m'appelle Ethan Lalienne, passionné d'informatique depuis mon plus jeune âge. Actuellement en BTS SIO option SLAM au lycée Paul Lapie, je poursuis désormais en Bachelor Développement Informatique à Sup de Vinci (La Défense), où j'approfondis le développement d'applications web et logicielles.

Mon parcours m'a conduit du bac STMG SIG jusqu'à des projets concrets en Java, PHP, Unity et développement de jeux vidéo. Je crois que la technologie doit servir des fins concrètes et j'aime concevoir des solutions utiles, bien construites et maintenables.

Passionné de basketball, musculation, et bien sûr de jeux vidéos — ce qui m'a naturellement conduit vers le game development comme projet personnel avec Unity.

First Car — Plateforme de réservation & App Java

2ème année
Janvier 2026 → Février 2026 · 5 semaines · Paris / Asnières (92)
Entreprise First Car — Transport VTC / Mise en relation chauffeurs
Adresse 60 rue François 1er, Paris (bureaux d'Asnières)
Tuteur M. Nicolas MORIN (PDG) & M. Hassan DJEDDOU (Associé)
Mission principale Plateforme de réservation en ligne (PHP/MySQL) + Application de gestion desktop (Java/Swing)

🎯 Réalisations semaine par semaine

🗄️
S1 — Analyse des besoins & conception BDD

Réunion de cadrage avec M. MORIN, interviews des chauffeurs et assistants. Conception du schéma relationnel MySQL (7 tables : users, cars, chauffeurs, chauffeur_availability, reservations, ratings, password_reset_tokens). Création via phpMyAdmin.

MySQLphpMyAdminMerise
🌐
S2 — Développement du site de réservation

Système d'authentification PHP (sessions, bcrypt, tokens CSRF). Page d'accueil dynamique depuis BDD. Système de réservation en 3 étapes : choix créneau → chauffeurs disponibles (requête SQL sur disponibilités) → confirmation + email automatique.

PHP/PDOSessionsbcryptCSRFEmail
⚙️
S3 — Espace administrateur

Dashboard avec statistiques (users, réservations, note moyenne). CRUD complet pour véhicules et chauffeurs (photos, langues, expérience, disponibilités par jour). Tableau des réservations filtrable par statut. Page de visualisation des avis clients (note 4,63/5).

CRUDPHPAdmin panelDashboard
☕
S4 — Application Java desktop

Application Swing avec VSCode. Architecture fichiers à la racine : DBConnection, User, Car, Chauffeur, Reservation, Session. Connexion MySQL via JDBC + connecteur 8.0.33. Hashage mots de passe JBCrypt. Interfaces : connexion, accueil, réservation (véhicule + chauffeur + date/heure + adresses), panel admin.

JavaSwingJDBCJBCryptVSCode
✅
S5 — Tests, corrections & documentation

Sessions de test avec M. MORIN et les assistants : connexion, réservation complète, historique. Corrections : format de date, bug affichage petits écrans, visibilité confirmation. Rédaction guide utilisateur. Livraison des fichiers à l'entreprise. Bilan final approuvé par M. MORIN.

Tests utilisateursDébogageDocumentation

💡 Compétences acquises

PHP / PDO / MySQL Java / Swing / JDBC Authentification sécurisée Conception BDD relationnelle CRUD & Admin panel Tests utilisateurs JBCrypt / bcrypt Documentation technique

📊 Bilan

Stage validé avec succès. Deux livrables opérationnels : site de réservation PHP/MySQL et application Java Swing. M. MORIN très satisfait — fichiers installés en production sur les ordinateurs de l'entreprise.
