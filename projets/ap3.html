🏦 Application Bancaire - AP3

📅 Décembre 2025 ⏱️ 70 heures 👨‍💻 Java, FXML & SceneBuilder

📖 À propos du projet

AP3 est une application de gestion bancaire développée en Java avec une interface graphique construite via FXML et SceneBuilder. L'application simule un système bancaire complet permettant de gérer des comptes, des transactions, et les informations des clients. Ce projet démontre l'application concrète des principes de la Programmation Orientée Objet (POO), notamment l'encapsulation avec des comptes bancaires sécurisés.

Ce projet m'a permis d'apprendre :

  • Les fondamentaux de la programmation orientée objet en Java (classes, héritage, encapsulation)
  • La création d'interfaces graphiques avec JavaFX, FXML et SceneBuilder
  • La persistance de données avec des fichiers ou bases de données légères
  • L'architecture d'une application Desktop avec modèle-vue-contrôleur (MVC)
  • La sécurité des données sensibles (mots de passe hachés, encapsulation)
  • La gestion des événements et des interactions utilisateur

🎯 Étapes principales du développement

1️⃣ Architecture et conception des classes

Conception de la structure objet de l'application : création des classes Client, CompteBancaire, Transaction avec leurs attributs privés et méthodes de gestion. Implémentation des principes de POO pour assurer une encapsulation robuste et une réutilisabilité du code.

🛠️ Technologies utilisées

  • Langage de programmation : Java 11+
  • Framework d'interface graphique : JavaFX
  • Outils de conception UI : SceneBuilder
  • Markup pour interface : FXML (XML)
  • Persistence des données : Sérialisation Java ou SQLite
  • IDE : Eclipse
  • Sécurité : Hachage de mots de passe (SHA256, MD5), Encapsulation stricte

✨ Compétences acquises

  • Maîtrise approfondie de la Programmation Orientée Objet (classes, encapsulation, héritage)
  • Développement d'applications Desktop avec JavaFX et FXML

  • Persistence et gestion des données
  • Implémentation de mesures de sécurité (authentification, encapsulation)
  • Créer des interfaces utilisateur professionnelles et intuitive
  • Gestion complète du cycle de vie d'une application Desktop
