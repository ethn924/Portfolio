📋 Plateforme de Gestion de Compte Rendu

Application web complète de gestion pédagogique développée en PHP/MySQL

📅 Jan. 2025 ⏱ 80 heures 🎓 AP1 – BTS SIO 🌐 PHP / MySQL
PHP MySQL HTML/CSS JavaScript PHPMailer MVC

Présentation du projet

Cette plateforme web est mon projet d'AP1 en BTS SIO. Il s'agit d'une application complète de gestion de comptes rendus pédagogiques développée en PHP avec une base de données MySQL. L'application permet aux élèves de soumettre leurs comptes rendus et aux professeurs de les corriger et commenter.

Fonctionnalités principales

👥

Système d'authentification

Inscription et connexion sécurisée pour élèves et professeurs avec vérification d'email.

📝

Gestion des comptes rendus

Soumission, modification et gestion complète des comptes rendus par les élèves.

🔍

Recherche avancée

Filtrage des comptes rendus par titre, matière, date ou statut.

✏️

Système de correction

Correction en ligne avec commentaires détaillés et notation par les professeurs.

📎

Upload de fichiers

Upload de documents (PDF, images, Word) avec chaque compte rendu.

🔔

Notifications email

Système de notifications via PHPMailer pour informer des corrections.

📊

Tableau de bord

Interface personnalisée pour élèves et professeurs avec statistiques et suivi.

🔐

Sécurité

Hachage des mots de passe, protection XSS, validation des données côté serveur.

Architecture technique

BDD

Base de données

Structure relationnelle optimisée avec tables pour utilisateurs, comptes rendus, commentaires, notifications et fichiers joints. Utilisation de clés étrangères et contraintes d'intégrité.

MVC

Architecture MVC

Séparation claire entre la logique métier (PHP), la présentation (HTML/CSS/JS) et les données (MySQL). Utilisation de fichiers include pour la réutilisabilité.

SEC

Sécurité

Protection contre les injections SQL via requêtes préparées, hachage sécurisé des mots de passe avec password_hash(), validation côté serveur.

Structure du projet

📁 PROJET_ETHAN_AP1/
├── 📄 index.php          — Page d'accueil
├── 📄 liste_cr.php       — Liste des comptes rendus
├── 📄 editer_cr.php      — Édition de compte rendu

├── 📄 ap1_ethan_2025.sql — Script SQL de la BDD
├── 📁 docs/              — Documentation et specs
└── 📁 uploads/           — Fichiers uploadés

Compétences développées

  • Backend : PHP procédural avancé, architecture d'application web, sessions utilisateur, envoi d'emails PHPMailer
  • Base de données : Conception de schéma relationnel, requêtes SQL complexes, transactions
  • Frontend : Interface utilisateur intuitive, JavaScript pour interactions dynamiques, responsive design
  • Sécurité : Protection XSS/SQL injection, authentification sécurisée, gestion des droits d'accès
