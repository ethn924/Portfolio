Développement Web - Projet Scolaire

📊 Application Web de Gestion de Stock

📅 Novembre 2025 ⏱️ 60 heures 👨‍💻 PHP & MySQL

📖 À propos du projet

Cette application web permet de gérer l'inventaire d'un magasin ou d'une entreprise. Elle a été développée en PHP avec une base de données MySQL. Le projet inclut une interface complète pour ajouter, modifier, supprimer et rechercher des produits, ainsi que la gestion des utilisateurs et des différents niveaux d'accès.

Ce projet m'a permis d'apprendre :

  • Les principes du développement web côté serveur (PHP)
  • La gestion des bases de données (MySQL) avec des requêtes SQL
  • Les opérations CRUD (Create, Read, Update, Delete)
  • La sécurité des applications web (authentification, autorisation)
  • La création d'interfaces utilisateur fonctionnelles et intuitives
  • La gestion des formulaires et de la validation des données

🎯 Étapes principales du développement

1️⃣ Configuration de la base de données

Création de la structure MySQL pour stocker les produits, les catégories, les utilisateurs et les mouvements de stock. Définition des relations et des contraintes pour assurer l'intégrité des données.

🛠️ Technologies utilisées

  • Langage de programmation : PHP 7.4+
  • Base de données : MySQL / MariaDB
  • Front-end : HTML5, CSS3, JavaScript
  • Sécurité : Password hashing (PHP password_hash), Prepared statements, Session management
  • Serveur : Apache / Nginx
