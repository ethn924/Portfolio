🌐 Application Web de Gestion de Stock

Application web de gestion d'inventaire développée en PHP/MySQL

📅 Nov. 2025 ⏱ 60 heures 🌐 PHP / MySQL
PHPMySQLHTML/CSSJavaScript

À propos du projet

Cette application web permet de gérer l'inventaire d'un magasin ou d'une entreprise. Développée en PHP avec une base de données MySQL, elle inclut une interface complète pour ajouter, modifier, supprimer et rechercher des produits, ainsi que la gestion des utilisateurs et des niveaux d'accès.

Fonctionnalités principales

📦

Gestion des produits

Ajout, modification, suppression de produits avec catégorie, prix, quantité et référence.

🔍

Recherche et filtres

Filtrage par catégorie, recherche par nom ou référence, tri par prix ou quantité.

👤

Gestion des utilisateurs

Comptes administrateur et employé avec niveaux d'accès différenciés.

📊

Suivi du stock

Alertes de stock faible, historique des mouvements, statistiques de l'inventaire.

🔐

Authentification

Système de connexion sécurisé avec sessions PHP et hachage des mots de passe.

📋

CRUD complet

Opérations Create, Read, Update, Delete sur tous les produits via interface web.

Étapes du développement

01

Base de données MySQL

Structure relationnelle pour produits, catégories, utilisateurs et mouvements de stock avec contraintes d'intégrité.

02

Authentification et sessions

Système de login sécurisé, gestion des sessions PHP et différenciation admin/employé.

03

Interface CRUD produits

Formulaires de saisie, tableau de liste avec actions, pagination et recherche en temps réel via JavaScript.

04

Sécurité et validation

Requêtes préparées PDO, validation serveur et client, protection contre les failles communes.

Compétences acquises

  • Développement web côté serveur avec PHP
  • Gestion de bases de données MySQL et requêtes SQL avancées
  • Opérations CRUD complètes
  • Authentification et sécurité web
  • Interfaces utilisateur fonctionnelles avec formulaires HTML
  • Validation côté serveur et client
