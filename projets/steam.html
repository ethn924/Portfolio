🔓 TP Phishing - Reproduction d'une Interface Steam

📅 Octobre 2025 ⏱️ 41 heures 👨‍💻 HTML, CSS & JavaScript

📖 À propos du projet

Ce projet est un travail pratique de cybersécurité ayant pour objectif de comprendre les mécanismes des attaques de phishing. J'ai reproduit l'interface du site Steam pour analyser comment les arnaqueurs se servent des sites familiers pour tromper les utilisateurs et récupérer des données sensibles.

Ce projet pédagogique m'a permis d'apprendre :

  • Les techniques utilisées dans les attaques de phishing
  • Comment reproduire fidèlement une interface web existante
  • Les vulnérabilités des utilisateurs face aux interfaces connues
  • Les bonnes pratiques pour reconnaître et se protéger contre le phishing
  • Les implications légales et éthiques de ces techniques
  • L'importance de la sensibilisation à la cybersécurité

🛡️ Signaux d'alerte pour détecter le phishing

  • URL suspecte : Vérifier l'adresse exacte du site (domaine, TLD)
  • Certificat SSL : Chercher le cadenas et "https://" dans la barre d'adresse
  • Demandes inhabituelles : Les vrais sites ne demandent jamais les identifiants par email
  • Design approximatif : Fautes d'orthographe, images de basse qualité
  • Liens cachés : Survolez les liens pour voir leur vraie destination
  • Emails suspects : Vérifier l'adresse expéditeur exacte

⚠️ Note éthique

Ce projet a été réalisé uniquement à des fins éducatives dans un contexte scolaire contrôlé. La création de vraies pages de phishing pour arnaquer les utilisateurs est illégale et contraire à l'éthique. L'objectif est de comprendre comment ces attaques fonctionnent pour mieux les prévenir et protéger les utilisateurs.
