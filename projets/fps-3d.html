🎯 FPS 3D - Mon Premier Projet Unity

📅 2024 - 2025 ⏱️ Projet en cours 👨‍💻 C# & Unity

📖 À propos du projet

Ce projet est mon premier vrai projet de game development. C'est un FPS (First-Person Shooter) 3D complet créé avec Unity et C#. J'ai suivi l'excellente série de tutoriels TUTO UNITY FR sur YouTube pour réaliser ce projet.

Ce projet m'a permis d'apprendre :

  • Les bases du développement 3D avec Unity
  • La gestion des contrôles et de la physique
  • Les systèmes de jeu (tir, santé, respawn)
  • L'interface utilisateur (UI)
  • La programmation orientée objet en C#

🛠️ Technologies utilisées

  • Unity Engine - Moteur 3D
  • C# - Langage de programmation
  • Mirror Networking - Pour le multijoueur
  • Physics Engine Unity - Physique 3D
  • Raycast System - Pour le système de tir

📚 Ce que j'ai appris

  • Programmation en C# et Programmation Orientée Objet
  • Développement 3D avec Unity (transforms, rigidbodies, colliders)
  • Système de jeu (game loops, event handling)
  • Programmation réseau avec Mirror
  • Gestion d'UI avec le système Canvas d'Unity
  • Optimisation et débogage
  • Architecture de code et design patterns
