📚 Système de Gestion de Bibliothèque

📅 Septembre 2025 ⏱️ 45 heures 👨‍💻 Java & Swing

📖 À propos du projet

Application de gestion de bibliothèque développée en Java avec une interface graphique Swing. Cette application permet de gérer complètement une bibliothèque : gestion des livres, des emprunteurs, des emprunts et des retours. Le projet intègre une base de données pour la persistance des données et des rapports de gestion.

Ce projet m'a permis d'apprendre :

  • La programmation orientée objet avancée avec Java
  • La création d'interfaces graphiques avec Swing
  • La connexion à une base de données depuis Java (JDBC)
  • L'implémentation de patterns de design (MVC, DAO)
  • La gestion d'événements et les interactions utilisateur
  • La génération de rapports et l'export de données

🎯 Étapes principales du développement

1️⃣ Architecture et modèle de données

Conception de l'architecture MVC avec les classes métier (Livre, Emprunteur, Emprunt), création de la base de données et implémentation des interfaces DAO pour l'accès aux données.

🛠️ Technologies utilisées

  • Langage : Java 11+
  • Interface graphique : Swing (JFrame, JPanel, JTable)
  • Base de données : MySQL / MariaDB
  • Connexion DB : JDBC avec prepared statements
  • IDE : Eclipse ou IntelliJ IDEA
  • Génération de rapports : JasperReports (optionnel)

✨ Compétences acquises

  • Développement complet d'applications desktop avec Java
  • Maîtrise de la POO et des patterns de design

  • Gestion des événements et interactions utilisateur
  • Intégration de bases de données relationnelles
