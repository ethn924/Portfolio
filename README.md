# Portfolio — Ethan Lalienne

Code source de mon portfolio personnel : [ethn924.github.io/Portfolio](https://ethn924.github.io/Portfolio/)

Étudiant en Bachelor Développement Informatique (Sup de Vinci) après un BTS SIO SLAM. Ce site présente mes projets (BTS, freelance, jeux vidéo), mon parcours, mon blog technique et mes stages.

## Structure

- `index.html` / `style.css` / `script.js` — site one-page
- `projets/` — fiche détaillée par projet
- `blog/` — articles techniques
- `assets/` — fichiers téléchargeables (CV, rapports, code source des projets)

## Stack

HTML · CSS · JavaScript vanilla — aucun framework, aucune dépendance de build.
